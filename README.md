# Курсовая работа
Курсовая работа по дисциплине "Разработка веб-приложений (ПИ) 2024" на тему "Система бронирования номеров в гостинице"

# Запуск
### Запуск бека из директории \backend\
```
npm i
npm start
```
### Запуск фронта из директории \frontend\
```
npm i
npm run dev
```
